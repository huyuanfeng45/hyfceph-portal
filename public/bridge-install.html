管理员 Chrome 扩展安装说明

这套扩展只装在管理员自己的电脑上。它会把你浏览器里当前可用的远程会话同步到 HYFCeph 门户，公开用户随后只需要在 skill 里发送侧位片即可。

给谁安装

只给管理员自己安装。公开用户不需要扩展，不需要分享链接，也不需要上游账号。

需要什么

Chrome 浏览器、门户管理员 API Key，以及部署包里的 chrome-extension/ 文件夹。

安装后作用

扩展会在你打开侧位片网页时自动同步可用会话，门户就能代用户完成远程测量。

H
安装步骤
只需管理员操作一次
1. 解压部署包 找到 hyfceph-portal/chrome-extension/

如果你手里是单独的扩展包，也可以直接解压扩展压缩包，然后使用其中的 chrome-extension/ 目录。

2. 打开扩展管理页 chrome://extensions/

打开右上角“开发者模式”。

3. 加载扩展 选择 chrome-extension/ 文件夹

点击“加载已解压的扩展程序”，选择整个扩展目录，而不是单个文件。

4. 生成管理员 API Key 在门户控制台里生成

必须使用管理员账号自己的 API Key，不要填普通用户的 Key。

5. 配置扩展 填门户地址和管理员 API Key

门户地址通常就是你部署后的正式域名，例如当前站点域名。

6. 同步一次 打开侧位片网页，再点“立即同步”

同步成功后，门户管理员面板里的“浏览器同步状态”会显示在线。
