Workflow

从注册到开始使用，一共四步。

界面不再堆信息，而是把每个角色最需要的动作放在最前面。

01

注册账号

填写名字、单位、手机号和密码，创建个人门户账号。

02

生成 Key

登录后在控制台生成个人 API Key，作为技能端调用凭证。

03

提交侧位片

在技能端直接上传侧位片，由服务端统一完成远程调用。

04

查看结果

获取 PNG 标注图、测量值和多套分析法数据，继续解读。
