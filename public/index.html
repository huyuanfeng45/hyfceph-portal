使用流程

从注册到开始调用，一共四步。

01

注册账号

填写名字、单位、手机号和密码，创建个人门户账号。

02

生成 API Key

登录控制台后生成个人 Key，作为技能端的调用凭证。

03

上传侧位片

在技能端直接提交侧位片，由服务端完成统一测量调用。

04

查看结果

接收 PNG 标注图、测量值与分析法结果，再继续选择分析法解读。
