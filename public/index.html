HYFCeph Access Portal

侧位片自动定点与测量的注册入口

HYFCeph 用于侧位片自动定点、常用头影测量值提取，以及面型分析的远程调用。 用户完成注册后即可登录控制台，生成自己的 API Key，随后把 API Key 提供给技能端即可开始使用。

适用场景

侧位片自动定点、头影测量、病例复核、临床教学和报告解读。

注册后获得

个人 API Key、登录控制台、Key 生成与重置能力。

推荐流程

注册账号，登录控制台，生成 API Key，然后在 HYFCeph 技能里提交使用。
